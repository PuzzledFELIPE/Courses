# Plano de Estudos - 30 Dias para Automação de Processos com Python

De iniciante a desenvolvedor capaz de automatizar processos reais.

## 🎯 Objetivo Geral
Dominar os fundamentos de Python e suas aplicações práticas em automação de processos, planilhas e manipulação de arquivos, preparando-se para trabalhar com RPA.

Ao final dos 30 dias, o resultado esperado é que você seja capaz de:
- Criar scripts Python que automatizam tarefas repetitivas em Excel.
- Manipular arquivos e organizar pastas do sistema automaticamente.
- Gerenciar versões do seu código com Git e GitHub.

## 💡 Dicas de Sucesso
- ❌ Erros são aprendizado – não desista!
- 🚀 Use o GitHub para seu portfólio.
- 📝 Documente seu código com comentários.
- 🧠 Tente fazer sozinho antes de olhar a resposta.
- 🔄 Pratique todos os dias.
- ⏱️ Dedique 1-2 horas diariamente.
